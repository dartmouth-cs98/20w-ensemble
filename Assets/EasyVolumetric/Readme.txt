Easy Volumetric is realtime lightweight solution for volumetric light, especially for mobile. All you need is configure light borders(window for example) and script will automatically handle all stuff. It's tested on android and showed great performance. You can adjust various settings like noise, physics, ray distance, color and etc.

To understand how its working please watch/read tutorials and open demo scene.

Components:
VolumetricProbe - main component. Place it on any object.

Other scripts:
VolumetricProbe - custom inspector for VolumetricProbe.
LightMovement - moves object. Used in demo scenes.
LightRotation - rotate transform. Used in demo scenes.
SceneSwitcher - switch scene if any key was pressed. Used in demo scenes.
